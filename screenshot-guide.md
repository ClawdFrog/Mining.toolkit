## üé® Design Tips for Screenshots

### Visual Style
- **Preferred:** Dark mode screenshots (matches underground use case)
- **Clean status bar:** Use Simulator or hide status bar
- **No personal info:** Make sure no real names/emails visible
- **High quality:** Use Simulator at 100% scale, then export

### Text/Labels
- You can add small text overlays in App Store Connect if needed
- Keep it minimal - let the app speak for itself

### Background
- App Store will add the iPhone/iPad frame automatically
- Or you can use tools like ShotBot or Screenshot Builder for fancy framing

## üõ†Ô∏è How to Capture

### Option 1: iOS Simulator (Recommended)
1. Open Xcode
2. Window ‚Üí Devices and Simulators
3. Choose iPhone 14 Pro Max (6.5")
4. Build and run app
5. Cmd+S to save screenshot
6. Screenshots save to Desktop

### Option 2: Physical Device
1. Run app on iPhone
2. Set up each screen
3. Press Side + Volume Up simultaneously
4. Transfer photos to Mac
5. Resize to correct dimensions if needed

### Required Dimensions:
- **iPhone 6.5":** 1284 x 2778 pixels (portrait)
- **iPhone 5.5":** 1242 x 2208 pixels (portrait)
- **iPad Pro:** 2048 x 2732 pixels (portrait)

## üí° Pro Tips

1. **Tell a story:** Order screenshots to show workflow (Home ‚Üí Calculators ‚Üí Results ‚Üí Drill Specs)

2. **Highlight offline:** Maybe add a screenshot with airplane mode on, showing it still works

4. **Localization:** If you have time, take screenshots in French and Spanish too for those App Store regions

5. **Test flight first:** Have a few miners test the app, then use the most impressive/useful screens
